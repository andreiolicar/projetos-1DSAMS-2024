## Verificador de Número Par ou Ímpar
**Este repositório contém um site simples que verifica se um número inserido pelo usuário é par ou ímpar. A aplicação foi desenvolvida utilizando JavaScript, HTML e CSS, e possui uma interface amigável e responsiva.**

**Funcionalidades:**

* Interface intuitiva: Design limpo e moderno, com um formulário fácil de usar.
* Verificação precisa: Determina corretamente se o número inserido é par ou ímpar.
* Animações sutis: Inclui animações para tornar a experiência do usuário mais agradável.
* Responsividade: Adapta-se a diferentes tamanhos de tela, proporcionando uma boa experiência em dispositivos móveis e desktops.
* Botão "Verificar Novamente": Permite que o usuário insira um novo número e realize a verificação novamente.

**Como executar o verificador:**

* Faça o download ou clone este repositório.
* Abra o arquivo index.html em seu navegador web.
  
**Tecnologias utilizadas:**

* HTML
* CSS
* JavaScript

**Observações:**

* O projeto inclui imagens que devem estar na pasta "imagens" para que funcione corretamente.
* O arquivo CSS (style.css) contém os estilos para a aparência do site.
* O arquivo JavaScript (script.js) contém a lógica para verificar o número e exibir o resultado.
